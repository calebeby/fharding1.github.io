GoPass (under dev)

GoPass is a client-server password manager that anyone can run, and while it is in development, it works pretty good so far (with limited features), and is fairly secure. By leveraging AES encryption, and encrypting client-side before sending it to the server, it's very hard for the plain text passwords to be stolen or decrypted by anybody besides someone with the key. Also, because it has a Go backend, and a PostgreSQL database the daemon should be fairly quick and be able to handle a decently sized infrastructure, especially after the code is cleaned up and optimized a bit.

Various RESTful webapps in Go

I've written many RESTful webapps in Go, and I love it. Go allows you to forget callback hell, and the POS known as NPM, and just use the stdlib to devlop very, very fast and maintainable RESTful (or not RESTful) apis/sites. Resource-cheap goroutines allow you to forget all that callback nonsense and write FAST easy to understand code.
